Opis stylu zewnętrznego

  1. Składnia teoretyczna → otoczony ramką (może być tabela jedno komórkowa, konieczne wcięcia).

    <head> ... <link rel="Stylesheet" type="text/css" href="style.css" /> ... </head>

  2. Gdzie wstawiany jest styl zewnętrzny, jaki znacznik jest za to odpowiedzialny oraz ile razy może on występować w dokumencie, jako podłączenie stylu CSS?

    Styl zewnętrzny wstawiany jest w sekcji nagłówka dokumentu, pomiędzy znacznikami <head> a </head>. Znacznik odpowiedzialny to <link rel="Stylesheet" />. Może on występować dowolną liczbę razy w dokumencie, aby dodać więcej niż jeden arkusz stylów.

  3. Jak jest najprostsza forma wstawiania stylu zewnętrznego → otoczony ramką (może być tabela jedno komórkowa?

    <link rel="Stylesheet" type="text/css" href="style.css" />

  4. Ile arkuszy styli może być dołączanych do jednego dokumentu HTML oraz jaka zasad musi być zachowana przy większej ilości dołączanych styli?

    Można dołączyć dowolną liczbę arkuszy stylów, każdy jako osobny znacznik <link rel="Stylesheet" />. Przy większej liczbie arkuszy, ważniejsze będą te, które są podłączone później.

  5. Co się dzieje w przypadku konfliktów? Przez konflikt rozumiemy, gdy te same style są definiowane w różnych arkuszach styli podłączone do tego samego arkusza.

    W przypadku konfliktów, style z arkusza podłączonego później będą miały wyższy priorytet i nadpiszą te z wcześniejszych arkuszy.

  6. Jak są tworzone i edytowane pliki arkuszy z użyciem edytora tekstowego?

    Pliki arkuszy stylów tworzymy i edytujemy jak zwykłe pliki tekstowe, używając prostego edytora tekstu, np. Notepad, a następnie zapisujemy je z rozszerzeniem .css.

  7. Jak są tworzone i edytowane pliki arkuszy z użyciem specjalnego edytora CSS?

    Pliki arkuszy stylów można tworzyć i edytować przy użyciu specjalnych edytorów CSS, takich jak Visual Studio Code, Sublime Text czy Brackets, które oferują narzędzia wspomagające pisanie kodu CSS.

  8. Jakie rozszerzenie posiadają plik będący zewnętrznym arkuszem styli?

    Plik będący zewnętrznym arkuszem stylów musi mieć rozszerzenie .css.

  9. Jakie są zalety zewnętrznego arkusza styli( dwie – z użyciem wyliczenia?

    1. Wszystkie strony w obrębie serwisu mogą mieć wspólne cechy stylów.
    2. Zmiany stylów, np. czcionki, można wprowadzać w jednym pliku .css, bez potrzeby edytowania każdej strony z osobna.

  10. Uwagi na temat pisania w CSS.

    Używaj odpowiednich selektorów, aby precyzyjnie określić, który element chcesz stylizować. Deklaracje stylów muszą kończyć się średnikiem. Grupy deklaracji w CSS powinny być otoczone nawiasami klamrowymi.

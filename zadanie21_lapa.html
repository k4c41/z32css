Pojęcie Znaczenie pojęcia
Co oznacza skrót ID? Skrót ID oznacza identyfikator danego elementu.
Jaki jest cel używania ID? ID umożliwia rozróżnienie podobnych elementów i przypisanie im indywidualnych stylów.
Jakie znaczenie ma znak #? Znak # poprzedza nazwę identyfikatora w arkuszu stylów CSS.
Jak wywołujemy ID (z wyjaśnieniem)? Wywołanie ID następuje przez atrybut id="nazwa" w znaczniku HTML, natomiast w CSS odwołujemy się do niego za pomocą #nazwa.
Ważne uwagi dotyczące stosowania ID (trzy punkty): 1. ID może występować tylko raz w całym dokumencie.
2. Każdy identyfikator powinien być unikalny.
3. W razie potrzeby wielokrotnego użycia lepiej zastosować klasę class.

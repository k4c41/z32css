Cechy modyfikowania formatowania w HTML oraz CSS

Cecha HTML CSS
Sposób formatowania Bezpośrednio w znacznikach HTML Osobne reguły w pliku .css lub w <style>
Separacja treści i stylu Brak – styl mieszany z treścią Pełna separacja treści i stylu
Elastyczność Ograniczona Duża (np. klasy i ID)
Możliwość ponownego użycia Ograniczona Styl można stosować do wielu elementów
Obsługa różnych urządzeń Brak wsparcia dla responsywności Media queries do różnych ekranów
Zaawansowane efekty Ograniczone Animacje, gradienty, cienie itp.
